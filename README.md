## About Void Studio

Void Studio is the only software studio in East Africa combining Backend, Mobile, AI, 3D, and DevOps under one roof — delivering complete digital products, not just code. Founded in 2023, we transform ideas from nothing into everything.

### What We Build

- **Digital Products** — End-to-end web and mobile product development
- **Brand & Design Systems** — Complete visual identities and UI/UX
- **AI-Powered Solutions** — Smart dashboards, recommendation engines, AI chatbots
- **Immersive 3D Experiences** — Interactive 3D visualizers and spatial interfaces
- **Mobile Applications** — Cross-platform iOS and Android apps
- **DevOps & Cloud** — CI/CD pipelines, containerized deployments, cloud scaling
- **Game Development** — Indie games, simulations, and gamified applications

## Getting Started

### Prerequisites

- Node.js 18+
- Bun (recommended) or npm/yarn

### Installation

```bash
# Install dependencies
bun install
# or
npm install
```

### Development

```bash
# Start development server
bun run dev
# or
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
# Production build
bun run build
# or
npm run build
```

### Preview Production Build

```bash
bun run preview
# or
npm run preview
```

### Testing

```bash
# Run tests once
bun run test
# or
npm run test

# Run tests in watch mode
bun run test:watch
# or
npm run test:watch
```

### Linting

```bash
bun run lint
# or
npm run lint
```
